Bookmarklets

One-click tools that load panel-api from CDN and inject powerful developer panels into any webpage. Each page shows the full source and a drag-to-bookmarks link.

ZUI Geometry Debugger
Wraps any page in a zoom/pan surface. Live cursor coordinates (screen + world), viewport state panel, and a geometry inspector that highlights hovered elements with bounding-box measurements.
Image Collector
Drag any image on a page onto the collection panel to harvest thumbnails and source URLs. One-click copy of all collected URLs.
Text Extractor
Click any element to capture its text, tag, and CSS selector. Accumulates entries in a panel. Export as JSON. Useful for scraping and refactoring workflows.
